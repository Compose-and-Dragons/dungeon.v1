## Curl Command

```bash
# Remove newlines from DATA 
DATA=$(echo ${DATA} | tr -d '\n')

JSON_RESULT=$(curl --silent ${BASE_URL}/chat/completions \
    -H "Content-Type: application/json" \
    -d "${DATA}"
)

CONTENT=$(echo "${JSON_RESULT}" | jq -r '.choices[0].message.content')
echo "${CONTENT}"
```
