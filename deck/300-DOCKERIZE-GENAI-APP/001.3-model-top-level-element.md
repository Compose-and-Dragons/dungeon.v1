## How to define models
```yaml
models:
  # simple definition
  embedding-model:
    model: ai/granite-embedding-multilingual:latest

  # more detailed definition
  non-player-character-model:
    model: ai/qwen2.5:1.5B-F16
    context_size: 4096
    runtime_flags:
      - "--temp"                # Temperature
      - "0.1"
      - "--verbose-prompt"
```
